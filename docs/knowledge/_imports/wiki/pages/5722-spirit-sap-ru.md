Spirit Sap is a Tier 2 Спиритизм Item that provides Bonus Health. Its active ability allows the user to target an enemy hero and reduce their Spirit Resist and Spirit Power for a short duration.

Spirit Sap is a Component of Волна безмолвия.

It can be purchased from the Shop for 1,600.
