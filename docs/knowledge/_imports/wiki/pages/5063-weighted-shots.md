Weighted Shots is a Tier 3 Weapon Item that can be purchased from the Shop for 3,200. Its component, Slowing Bullets, makes up 1,600 of the cost.

This effect helps mitigate crowd-control by shortening stuns' durations, allowing the hero to recover more quickly.

## Buildup Per Shot

The item has a different buildup per shot depending on which hero purchased the item. The status only triggers once a buildup reaches 100% on an enemy player, and the buildup decays back to 0 if an enemy hasn't been shot for some time. Melee attacks count as 1 shot.
