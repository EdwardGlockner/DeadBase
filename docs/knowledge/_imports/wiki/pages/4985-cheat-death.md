## Trivia

- Cheat Death has the same name and similar effect to one of the abilities of the unreleased hero Rutger.

- Cheat Death's icon has been redesigned multiple times: Originally it showed a hand holding 5 aces in a poker game against Death, then as a person kicking the door of a coffin, and currently is a hand holding 5 aces including a skull suit.
