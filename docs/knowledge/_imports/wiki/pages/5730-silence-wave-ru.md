Silence Wave is a Tier 3 Спиритизм Item that provides Bonus Health to the purchasing hero. It was previously named Silence Glyph.

Silence Wave has an active ability that fires an expanding projectile which Silences any target it pierces, as well as reducing their Spirit Resist and Spirit Power for a longer duration.

It can be purchased from the Shop for 3,200, with its Component, Спиритическое истощение, making up 1,600 of the cost.
