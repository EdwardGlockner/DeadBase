Slowing Bullets is a Tier 2 Weapon Item that can be purchased from the Shop for 1,600.

It is a component of Weighted Shots.

## Buildup Per Shot

The item has a different buildup per shot depending on which hero purchased the item. The status only triggers once a buildup reaches 100% on an enemy player, and the buildup decays back to 0 if an enemy hasn't been shot for some time. Melee attacks count as 1 shot.
