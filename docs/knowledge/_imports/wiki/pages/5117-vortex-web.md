Vortex Web is a Tier 4 Spirit Item that can be purchased from the Shop for 6,400. Its component, Slowing Hex, makes up 1,600 of the cost.

- Movement Silence removes the knockback movement from Shiv's alt-fire.

## Hero abilities silenced by Vortex Web
