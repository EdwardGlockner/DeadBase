Crouching is a basic action available to the whole cast.

When crouching, you’ll nearly instantly decelerate to 4.8 m/s, stopping momentum. This can be used for more efficient direction changes while moving (tap crouch and change direction).

When moving over 8.9 m/s or 350 u/s, you will instead slide.
