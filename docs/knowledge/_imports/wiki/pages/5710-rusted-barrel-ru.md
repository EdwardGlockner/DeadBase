Rusted Barrel is a Tier 1 Спиритизм Item that provides Bonus Health.

It has an active ability which, when used against a target enemy, applies a Fire Rate Slow.

Rusted Barrel is a component of Чары обезоруживания.

It can be purchased from the Shop for 800.
