## Flame Dash Slide

Flame Dash is eligible to be cancelled at any time during its duration. By cancelling, you are able to carry over some momentum from Flame Dash for a short timeframe.

During that time frame, you can execute a slide that uses the momentum from the cancelled Flame Dash. This is called a Flame Dash Slide.

To execute, cancel Flame Dash and immediately slide.

## Flame Dash Slide Bunnyhopping

To start bunnyhopping after a Flame Dash Slide, you have to release the duck key while in the air and then quickly repress it again.

This is because Flame Dash Slide is not eligible for bunnyhopping, and you need Slide Storage to be able to bunnyhop.

To execute, release the duck key while in the air, then press it again to be able to bunnyhop.
