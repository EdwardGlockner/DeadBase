Heroic Aura is a Tier 3 Оружие Item that grants Sprint Speed to the purchasing hero, along with a boost to Bullet Resist for the buyer and nearby friendly Heroes and Minions.

Heroic Aura has an Active component which can be used to provide bonus Move Speed and Fire rate to yourself and nearby allies for a short time. Friendly Minions get double value from this buff.

It can be purchased from the Shop for 3,200.
