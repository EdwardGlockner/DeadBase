Active Reload is a Tier 2 Weapon Item that can be purchased from the Shop for 1,600.

## Notes

- If the player presses the reload button while outside the highlighted section, the item will fail to reload. The item only goes into cooldown once the effect is activated.

- The duration of the Active Reload activation window is always 0.3s. The highlighted section changes size proportionately to the hero's Reload Time.
