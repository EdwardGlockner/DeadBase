This article is part of the Deadlock Movement Wiki.

Item Tech is movement tech that is only available by buying a specific item from the shop. Usually, item tech is performed with an active item that is movement related, although there are some exceptions.

Below are a list of all item tech and movement relevant items in the game:

- Capacitor

- Majestic Leap Cancel

- Rescue Beam Slingshot

- Slowing Hex

- Stamina Mastery

- Vortex Web

- Warp Dashing

- Warp Dash Jump
