Extended Magazine is a Tier 1 Zbraň Item that grants more Ammo and more Weapon Damage. It was previously named Basic Magazine.

Extended Magazine is a Tier 1 Zbraň Předmět that can be purchased from the Shop for 800.

It is a component of Eskalující odolnost a Obří zásobník.
