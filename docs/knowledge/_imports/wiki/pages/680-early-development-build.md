The Early Development Build is the closed beta1] of Deadlock. It possesses a wide array of placeholder designs and models. In order to access the build, one must first receive an invite from someone who already has access.

The official Deadlock forums were created on April 3, 2024, and the first posts on the forum were made on April 16.2] The official Discord server was created on April 20. Matchmaking was opened for the first time on April 24.3]

Players can create a forum account and join the Discord server through the Player Feedback button in the lower-left corner of the main menu (Esc by default).

During the beta, feedback is heavily encouraged. However, due to developer concerns about echo chambers of opinions being formed, it is best that all feedback be given via the game's official forums. The forums have private sections for different kinds of feedback, in which users can only view their own posts. This helps the developers better identify points where several players have independently given the same feedback.

Players can also report bugs, suggest new features, heroes and items for the game via the public forum sections, some of which have been added, such as Magic Carpet.

## Experimental Build

In May 2025, a second private Deadlock build was accidentally shared in a livestream, revealing then-upcoming new heroes and redesigns for existing heroes.4]
