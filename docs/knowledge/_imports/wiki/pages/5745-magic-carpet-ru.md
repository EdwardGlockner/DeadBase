Magic Carpet is a Tier 4 Спиритизм Item that grants Ability Duration, Spirit Power and Bonus Health.

Magic Carpet has an active ability which can be used to summon a "Magic Carpet" that lets the using hero fly.

It can be purchased from the Shop for 6,400.
