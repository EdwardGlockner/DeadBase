Universal Tech is movement tech that more than one character in the game can execute. These techniques are usually executable by most or all of the cast, though some involve specific interactions with a handful of characters.

Note: Some movement tech articles use source units per second (u/s) instead of meters per second. Source units in Deadlock are equivalent to inches, and thus can be converted to meters at a rate of 39,37:1. The player's speed in units can be viewed with the console command cl_showpos 1.
