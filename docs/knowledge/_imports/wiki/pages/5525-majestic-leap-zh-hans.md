Majestic Leap is a Tier 3 活力 Item that allows the user to launch themselves high into the air, using the item again while in air allows the user to drop faster.

It can be purchased from the Shop for 3,200.

## Notes

After using the active again while in the air, you can use an Air Jump to travel further.
